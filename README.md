🚀 **Live Demo:** [View Live Portfolio](https://cheerful-marshmallow-c4f636.netlify.app/)

## 🎨 Creative Engineering & UI Experience

This portfolio isn't just a static resume; it's a showcase of interactive frontend engineering. 
Built from scratch using modern vanilla web standards and powered by high-performance animation engines.

### Key Features:
* **GSAP Scroll Trigger Engine:** Fluid, scroll-driven staggered layouts for Process and Services sections that adapt smoothly without breaking layout flows.
* **Smart Device Responsiveness:** Dynamic screen coordinate handling that optimizes execution for touch-screens and mobile URL-bar resizing anomalies.
* **Custom Dynamic Cursor:** A fluid, lerp-based physics cursor that expands contextually when hovering over interactive elements.
* **System-Aware Theme Toggle:** LocalStorage-backed native Light/Dark mode switcher for persistent user preference.
* **Asynchronous Serverless Contact Form:** Fully functional ajax-driven contact form hooked up with FormSubmit for direct email delivery without client-side page hijacking.

## 🛠️ Built With

* **Markup & Core Logic:** Native Semantic HTML5 & Modern ES6+ JavaScript.
* **Styling Architecture:** Clean CSS3 with CSS Grid, Flexbox, Custom Properties (Variables), and custom cubic-bezier transitions.
* **Animation Core:** [GSAP (GreenSock Animation Platform)](https://gsap.com/) & ScrollTrigger Plugin.
* **Form Processing:** FormSubmit API.
